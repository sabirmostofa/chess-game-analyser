Chrome Extension

Privacy Policy

Last updated: March 3, 2026

This privacy policy applies to the Chess.com -> Lichess One-Click Import extension.

Overview

The extension helps you import public Chess.com games into Lichess analysis. It operates locally in your browser and only sends data needed to perform that function.

Information We Store Locally

  • chessComUsername is stored locally in chrome.storage.local so the extension can run in one click.
  • manualSelectionEnabled is stored locally in chrome.storage.local to remember whether manual game picker mode is enabled.

Information Processed In Memory

  • When manual selection mode is used, the extension fetches your recent public Chess.com games and displays up to 10 entries in memory for selection.
  • PGN data from the selected game is processed only to open that game on Lichess analysis.
  • The recent-games list and selected PGN are not persisted by the extension beyond normal browser/page runtime.

Network Requests and Third-Party Services

The extension sends network requests only to:

  • https://api.chess.com/* to fetch your public game archives and PGN.
  • https://lichess.org/* to import/open analysis for the selected game.

Chess.com and Lichess operate as independent services under their own privacy policies and terms.

Purpose and Legal Basis

  • Purpose: provide game import and analysis-opening functionality.
  • Legal basis (where applicable): user-requested service performance and legitimate interest in operating the extension.

Data Sharing and Sale

  • No user data is sold.
  • No user data is shared with third parties beyond the API requests required for core functionality.

Retention

  • Stored settings remain in chrome.storage.local until you change them or remove the extension data.
  • In-memory game lists/PGN are not retained after runtime ends.

Security

We use browser-provided storage and HTTPS requests. No system can be guaranteed 100% secure, but reasonable measures are used to reduce risk.

International Data Transfers

API endpoints may be hosted in different countries. By using the extension, network requests may involve cross-border data transfer via those services.

Remote Code

The extension does not download and execute remote code.

Changes to This Policy

We may update this policy from time to time. The "Last updated" date at the top indicates the latest version.
