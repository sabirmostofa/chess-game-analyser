Chrome Extension

Chess.com -> Lichess One-Click Import

Import your latest Chess.com game to Lichess analysis with one click, or manually pick from your last 10 games.

Privacy Policy GitHub Repository

Features

  • One-click latest game import from toolbar icon.
  • Manual selection mode with last 10 recent games.
  • Game list includes opponent, color, and result.
  • Username validation against Chess.com in settings.
  • Lichess import flow with fallback to compact PGN URL.

Install

  1. Open chrome://extensions.
  2. Enable Developer mode.
  3. Click Load unpacked.
  4. Select this extension folder.
  5. Pin the extension from the Chrome Extensions menu.

First-Time Setup

  1. Go to chrome://extensions -> this extension -> Extension options.
  2. Enter your Chess.com username and save.
  3. Optional: enable manual selection mode.

How It Works

  1. Click the extension icon.
  2. If manual mode is off, latest game imports directly. If on, choose one from the last 10 games.
  3. Selected game opens on Lichess analysis.

Notes

  • Only public Chess.com games can be imported.
  • Uses Chess.com public API and Lichess web import flow.
  • Popup appears only if username is missing/invalid or API/import fails.
